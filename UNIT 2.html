I have done another project were I have worked with spreadsheets in many ways from doing calulation to creating a spreadsheet for a company, this contained many things such as pages and calulation which were made easy due to features that MS Excel offered. The facilities of spreadsheets make calculations easier to understand by displaying stages of the development of a sum, with the ability to write notes explaining each line. The grid encasing the cells in the spreadsheet can either be visible or hidden in print outs, enabling reports to be formatted from the display of the spreadsheet.Another good use of spreadsheets which I have used and found useful is Spreadsheet programs can transform data sets into various types of graphical displays. In Excel, the Charts section provides the ability to present a data set within a chart or graph. This allows users to visualize and communicate their data sets within presentations, for example using pie or bar charts. Visualizing data in this way can help you understand the data and inform future planning decisions.
